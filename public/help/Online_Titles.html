 Online Titles

Use the Online Titles tab to access the following menu options (see test link):

·           Online Titles: view and create all online titles.

·           Game Environments: view all game environments and create new layouts and/or environments.

·           Recent Games: quickly retrieve environments that you have managed and/or viewed during your current login. The list restarts with each new login session.
